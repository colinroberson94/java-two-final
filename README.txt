IDE Version Number: IntelliJ IDEA 2021.1.3 (Community Edition)

Directions: The application can be run by pressing the "Run" button at the top of the IntelliJ IDE. After the program starts, a login screen will be shown.
            Upon entering valid login information, the main application functionality will be presented. Here users can Create/Read/Update/Delete customer
            records, appointment records, and view select reports.

Additional Report: An additional report titled "Total appointments by customer" that shows the total number of appointments broken down by customer has been added.

MySQL driver version: mysql-connector-java-8.0.25
